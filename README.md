# Learn C++ game server

跟着[《多人在线游戏架构实战：基于C++的分布式游戏编程》](https://book.douban.com/subject/35260126/) 学习 C++ 游戏服务器开发。

## Build

Install [XMake](https://xmake.io/#/guide/installation).

```shell
xmake # build all
```
